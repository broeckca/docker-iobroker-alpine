# IoBroker  

Docker image for http://www.iobroker.net  

Base on node:6-alpine, iobroker v1.0.3
Minimal package installed

# Run IoBroker
```
$ docker run --name iobroker-alpine -d -p 8081:8081 -it salenss/iobroker-alpine
```
